## 📖 Project Overview

This project is a Digital Literacy Portfolio created as part of the Digital Literacy course. The aim of this project is to understand and demonstrate essential digital skills required in today’s world.

The project consists of five tasks covering topics such as digital awareness, professional online presence, coding platforms, communication etiquette, and cybercrime awareness.

All the tasks are organized in this GitHub repository along with screenshots, documents, and explanations.

---

## 🎯 Objectives

- To understand the importance of digital literacy
- To create and manage professional online profiles
- To explore coding and collaboration platforms
- To learn proper email communication etiquette
- To understand cybercrime and online safety
